# 🔐 Password Strength Analyzer

A modern full-stack web application that analyzes password strength, generates secure passwords, and detects password reuse using hashing for better security.

---

## 🚀 Features

- 🔍 Password strength checker (real-time analysis)
- 📊 Strength scoring system (Weak / Medium / Strong)
- 🔐 Secure password hashing using SHA-256
- 🔁 Password reuse detection system
- 🎲 Strong password generator
- 💡 Smart improvement suggestions
- 🎨 Modern glassmorphism UI design
- 📱 Fully responsive (mobile + desktop friendly)

---

## 🛠️ Tech Stack

- Python 🐍
- Flask 🌐
- HTML5
- CSS3 (Glassmorphism UI)
- Jinja2 Templates
- hashlib (SHA-256 security)
