# 🔐 Password Strength Analyzer (CLI Tool)

A Python-based command-line application that checks password strength, generates secure passwords, and prevents password reuse using SHA-256 hashing.

---

## 🚀 Features

- Password strength checker (Weak / Medium / Strong)
- Score-based evaluation system (0–5)
- Security suggestions for weak passwords
- Strong password generator
- Password reuse detection using SHA-256 hashing
- Stores password history in a local file
- Interactive CLI menu system

---

## 🧠 Concepts Used

- Python functions
- Loops and conditionals
- File handling
- Exception handling
- SHA-256 hashing (cryptography)
- Random password generation
- CLI (Command Line Interface)
